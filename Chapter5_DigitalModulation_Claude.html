MODULE 1 Geometric Representation of Signal Waveforms

1.1 The Big Picture

After A/D conversion, we have a stream of binary digits. But physical channels (wires, air) carry analog waveforms, not bits. So we need to map digital info → analog signal waveforms.

Baseband Channel

Frequency passband includes \(f = 0\). No carrier needed. Example: wired Ethernet.

Bandpass Channel

Passband far from \(f = 0\). Must modulate a carrier \(\cos(2\pi f_c t)\). Example: WiFi, cellular.

  • Binary Modulation: Each bit → one of 2 waveforms \(s_1(t)\) or \(s_2(t)\).
  • M-ary Modulation: Each block of \(k\) bits → one of \(M = 2^k\) distinct waveforms \(s_m(t)\), \(1 \le m \le M\).

To analyze these waveforms efficiently, we develop a vector representation — turning continuous signals into points in a finite-dimensional space. This dramatically simplifies the analysis of error probability.

1.2 Gram-Schmidt Orthogonalization

🧠 Physical Intuition: Imagine decomposing a 3D force vector into \(x\), \(y\), and \(z\) components. Gram-Schmidt does the same thing for signal waveforms — it finds the fundamental "axes" (orthonormal basis functions) of the signal space.

Given \(M\) signal waveforms, we construct \(N \le M\) orthonormal waveforms \(\{\psi_n(t)\}\):

Step 1: First basis function

\[\psi_1(t) = \frac{s_1(t)}{\sqrt{\mathcal{E}_1}}, \quad \mathcal{E}_1 = \int_{-\infty}^{\infty} s_1^2(t)\,dt\]

We simply normalize \(s_1(t)\) to have unit energy.

Step 2: For each subsequent signal \(s_k(t)\)

Compute how much of \(s_k(t)\) lies along each existing basis function (the projections):

\[c_{ki} = \int_{-\infty}^{\infty} s_k(t)\psi_i(t)\,dt, \quad i = 1, 2, \ldots, k-1\]

Subtract out those projections to get the genuinely new part:

\[d_k(t) = s_k(t) - \sum_{i=1}^{k-1} c_{ki}\psi_i(t)\]
  • If \(d_k(t) \neq 0\): normalize it → \(\psi_k(t) = d_k(t)/\sqrt{\mathcal{E}_k}\), where \(\mathcal{E}_k = \int d_k^2(t)\,dt\).
  • If \(d_k(t) = 0\): the signal is a linear combination of existing basis functions. No new dimension is added.

Key Properties of the Representation

  • \(N\) = dimensionality of the signal space. \(N = M\) if all signals are linearly independent.
  • Each signal is an exact linear combination: \(s_m(t) = \sum_{n=1}^{N} s_{mn}\psi_n(t)\)
  • Coefficients: \(s_{mn} = \int_{-\infty}^{\infty} s_m(t)\psi_n(t)\,dt\)
  • Energy = squared vector length: \(\mathcal{E}_m = \sum_{n=1}^{N} s_{mn}^2 = ||\mathbf{s}_m||^2\)
  • Inner product preserved: \(\int s_m(t)s_n(t)\,dt = \mathbf{s}_m \cdot \mathbf{s}_n\)
  • The basis set is NOT unique, but distances and energies are invariant!
WORKED EXAMPLE 1

Given 4 signal waveforms, find the orthonormal basis and vector representation.

Computing \(\psi_1(t)\):

\[\mathcal{E}_1 = \int_0^2 1^2\,dt = 2, \quad \psi_1(t) = \frac{s_1(t)}{\sqrt{2}}\]

Computing \(\psi_2(t)\):

\[c_{21} = \int_{-\infty}^{\infty} s_2(t)\psi_1(t)\,dt = \int_0^1 \frac{1}{\sqrt{2}}dt + \int_1^2 \frac{-1}{\sqrt{2}}dt = 0\]

Since \(c_{21} = 0\), \(s_2(t)\) is already orthogonal to \(\psi_1(t)\). So \(d_2(t) = s_2(t)\), \(\mathcal{E}_2 = 2\), and \(\psi_2(t) = s_2(t)/\sqrt{2}\).

Computing \(\psi_3(t)\):

\[c_{31} = 0, \quad c_{32} = -\sqrt{2}\] \[d_3(t) = s_3(t) - (-\sqrt{2})\psi_2(t) = s_3(t) + \sqrt{2}\psi_2(t)\] \[\mathcal{E}_3 = 1 \implies \psi_3(t) = d_3(t)\]

Computing \(\psi_4(t)\):

\[c_{41} = \sqrt{2}, \quad c_{42} = 0, \quad c_{43} = 1\] \[d_4(t) = s_4(t) - \sqrt{2}\psi_1(t) - \psi_3(t) = 0\]

Since \(d_4(t) = 0\), no new basis function! \(N = 3\) dimensions for 4 signals.

Vector representations:

\[\mathbf{s}_1 = (\sqrt{2},\, 0,\, 0), \quad \mathbf{s}_2 = (0,\, \sqrt{2},\, 0)\] \[\mathbf{s}_3 = (0,\, -\sqrt{2},\, 1), \quad \mathbf{s}_4 = (\sqrt{2},\, 0,\, 1)\]

With an alternative basis, the same signals become: \(\mathbf{s}_1 = (1,1,0)\), \(\mathbf{s}_2 = (1,-1,0)\), \(\mathbf{s}_3 = (-1,1,1)\), \(\mathbf{s}_4 = (1,1,1)\). Different vectors, same geometric relationships!

MODULE 2 Binary Modulation Schemes

Two fundamental categories for binary (1 bit at a time) signaling:

1. Antipodal Signaling

\(s_2(t) = -s_1(t)\). Signals are mirror images. 1D representation. Most power-efficient binary scheme.

2. Orthogonal Signaling

\(\int s_1(t)s_2(t)\,dt = 0\). Signals are perpendicular in signal space. 2D representation.

2.1 Binary PAM (Antipodal)

🧠 Intuition: Like a light switch with +V and -V. Bit '1' sends a positive pulse, bit '0' sends the negative version.

\[s_m(t) = A_m \cdot g_T(t), \quad 0 \le t \le T_b, \quad m = 1, 2\]

where \(A_1 = A\), \(A_2 = -A\), and \(g_T(t)\) is a rectangular pulse of unit amplitude and duration \(T_b\).

  • Bit rate: \(R_b = 1/T_b\) bits/sec
  • Both waveforms have equal energy: \(\mathcal{E}_b = A^2 T_b\), so \(A = \sqrt{\mathcal{E}_b / T_b}\)
  • Basis function: \(\psi(t) = g_T(t)/\sqrt{T_b}\) (unit energy rectangular pulse)

Geometric representation (1D):

\[s_1 = +\sqrt{\mathcal{E}_b}, \quad s_2 = -\sqrt{\mathcal{E}_b}\]

Distance between points: \(d_{12} = 2\sqrt{\mathcal{E}_b}\). Any pair of antipodal waveforms — regardless of their specific shape — has this same geometry!

2.2 Binary ASK

Binary PAM signals modulate a carrier: \(s_{1,2}(t) = \pm p(t)\cos(2\pi f_c t)\). For a rectangular pulse with \(p(t) = \sqrt{2\mathcal{E}_b/T_b}\):

\[\psi(t) = \sqrt{\frac{2}{T_b}}\cos(2\pi f_c t), \quad 0 \le t \le T_b\]

The energy of each waveform is \(\mathcal{E}_b\). The geometric representation is identical to Binary PAM — still 1D antipodal!

The only difference: the spectrum is now centered around \(\pm f_c\) (bandpass) instead of \(f = 0\) (baseband).

2.3 Binary PPM (Pulse Position Modulation)

🧠 Intuition: Same pulse shape, different time slot. '1' puts the pulse in the first half; '0' puts it in the second half.

The two pulses are orthogonal (they don't overlap in time) and have equal energy \(\mathcal{E}_b\):

\[\int_0^{T_b} s_1(t)s_2(t)\,dt = 0\]

Geometric representation (2D):

\[\mathbf{s}_1 = (\sqrt{\mathcal{E}_b},\, 0), \quad \mathbf{s}_2 = (0,\, \sqrt{\mathcal{E}_b})\]

PPM is a baseband orthogonal signaling scheme.

2.4 Binary FSK (Frequency Shift Keying)

🧠 Intuition: Different bits → different frequencies. '1' transmits a cosine at \(f_1\), '0' transmits at \(f_2\).

\[s_1(t) = \sqrt{\frac{2\mathcal{E}_b}{T_b}}\cos(2\pi f_1 t), \quad s_2(t) = \sqrt{\frac{2\mathcal{E}_b}{T_b}}\cos(2\pi f_2 t)\]

where \(f_1 = k_1/(2T_b)\), \(f_2 = k_2/(2T_b)\) for distinct positive integers \(k_1, k_2\). Minimum frequency separation:

\[\Delta f = f_2 - f_1 = \frac{m}{2T_b}, \quad m = |k_2 - k_1|\]

The signals are orthogonal with equal energy → same geometric representation as PPM!

\[\mathbf{s}_1 = (\sqrt{\mathcal{E}_b},\, 0), \quad \mathbf{s}_2 = (0,\, \sqrt{\mathcal{E}_b})\]

FSK is a bandpass orthogonal signaling scheme.

MODULE 3 Optimum Receiver & Error Probability

3.1 The AWGN Channel

The received signal is: \(r(t) = s_m(t) + n(t)\), where \(n(t)\) is AWGN with power spectral density \(S_n(f) = N_0/2\) W/Hz.

Our goal: design a receiver that minimizes the probability of error (\(P_E\)).

3.2 Receiver Architecture

  1. Signal Demodulator: Converts \(r(t)\) → observation vector \(\mathbf{y}\), same dimensionality as the signal space.
  2. Detector: Decides which of the \(M\) waveforms was transmitted based on \(\mathbf{y}\).

3.3 Two Types of Demodulators

Correlation-Type (Coherent)

Multiplies \(r(t)\) by each basis function \(\psi_n(t)\) and integrates. Requires phase synchronization. High accuracy but complex.

Matched Filter

Filters \(r(t)\) with a filter matched to the signal shape, then samples the output. Simpler to implement but can be more noise-prone.

3.4 Error Probability — The Key Results

The general formula ties error probability to the Euclidean distance between signal points:

\[\boxed{P_E = Q\!\left(\sqrt{\frac{d_{12}^2}{2N_0}}\right)}, \quad \text{where } d_{12}^2 = \int_{-\infty}^{\infty}|s_1(t) - s_2(t)|^2\,dt = ||\mathbf{s}_1 - \mathbf{s}_2||^2\]
Scheme\(d_{12}^2\)\(P_E\)
Binary Antipodal (PAM/BPSK) \(4\mathcal{E}_b\) \(Q\!\left(\sqrt{\dfrac{2\mathcal{E}_b}{N_0}}\right)\)
Binary Orthogonal (FSK/PPM) \(2\mathcal{E}_b\) \(Q\!\left(\sqrt{\dfrac{\mathcal{E}_b}{N_0}}\right)\)
⚡ CRITICAL INSIGHT

Antipodal signaling requires 3 dB (factor of 2) less energy than orthogonal signaling for the same error probability. This is because the antipodal distance \(d_{12}=2\sqrt{\mathcal{E}_b}\) is larger than the orthogonal distance \(d_{12}=\sqrt{2\mathcal{E}_b}\) for the same energy per bit.

WORKED EXAMPLE 4

Determine the bit error probability for the piecewise signals:

\[s_1(t) = \begin{cases} \sqrt{2} & 0 \le t < T_b/2 \\ 1 & T_b/2 \le t < T_b \end{cases}, \quad s_2(t) = \begin{cases} 2t/T_b & 0 \le t < T_b \\ 0 & \text{otherwise} \end{cases}\]

Solution: Compute the squared Euclidean distance:

\[d_{12}^2 = \int_0^{T_b}|s_1(t) - s_2(t)|^2\,dt = \frac{4}{3}T_b\] \[\boxed{P_E = Q\!\left(\sqrt{\frac{d_{12}^2}{2N_0}}\right) = Q\!\left(\sqrt{\frac{2T_b}{3N_0}}\right)}\]

Takeaway: You don't need to find the basis functions or classify the modulation type. Just compute the distance!

MODULE 4 M-ary Digital Modulations

Instead of transmitting 1 bit at a time, we group \(k\) bits into a symbol and select from \(M = 2^k\) waveforms.

  • Symbol rate: \(R_s = 1/T\) symbols/sec
  • Bit rate: \(R_b = k \cdot R_s = k/T = 1/T_b\)
  • Symbol energy vs. bit energy: \(\mathcal{E}_s = k\mathcal{E}_b = (\log_2 M)\mathcal{E}_b\)

4.1 M-ary PAM & M-ary ASK

🧠 Intuition: Instead of just +A and -A, use \(M\) different amplitude levels. All signals share the same pulse shape — it's a 1D, amplitude-only scheme.

\[s_m(t) = A_m \cdot p(t) = s_m \cdot \psi(t), \quad m = 1, 2, \ldots, M\]

To minimize average energy and avoid DC offset, amplitudes are symmetric about the origin and equally spaced:

\[A_m = (2m - 1 - M)\,A, \quad m = 1, 2, \ldots, M\]

For \(M = 4\): amplitudes are \(\{-3A,\, -A,\, +A,\, +3A\}\).

Average Energy

\[\mathcal{E}_{av} = \frac{A^2\mathcal{E}_p}{M}\sum_{m=1}^{M}(2m-1-M)^2 = \frac{A^2\mathcal{E}_p(M^2-1)}{3}\]

Signal constellation points: \(s_m = (2m-1-M)d\), where \(d = A\sqrt{\mathcal{E}_p}\).

M-ary ASK (Bandpass Version)

Modulate the PAM signal onto carrier \(\cos(2\pi f_c t)\). When \(f_c \gg W\), bandpass energy = 1/2 × baseband energy.

\[\psi_c(t) = \sqrt{\frac{2}{\mathcal{E}_p}}\,p(t)\cos(2\pi f_c t), \quad N = 1\]

Error Probability (M-ary PAM/ASK)

\[\boxed{P_M = \frac{2(M-1)}{M}\,Q\!\left(\sqrt{\frac{6\mathcal{E}_{av}}{(M^2-1)N_0}}\right) = \frac{2(M-1)}{M}\,Q\!\left(\sqrt{\frac{6(\log_2 M)\,\mathcal{E}_{b,av}}{(M^2-1)N_0}}\right)}\]
WORKED EXAMPLE 5

For \(M = 4\) PAM with amplitudes \(\{-3A,\,-A,\,+A,\,+3A\}\) and rectangular pulse of duration \(T\):

\[\mathcal{E}_{av} = \frac{1}{4}(9A^2T + A^2T + A^2T + 9A^2T) = 5A^2T = 5d^2\]

4.2 Phase-Shift Keying (M-PSK)

🧠 Intuition: Keep the amplitude constant but rotate the phase. All \(M\) signal points sit on a circle in 2D signal space — like hours on a clock face.

\[u_m(t) = p(t)\cos\!\left(2\pi f_c t + \frac{2\pi m}{M}\right), \quad m = 0, 1, \ldots, M-1\]

For a rectangular pulse \(g_T(t)\) with amplitude \(\sqrt{2\mathcal{E}_s/T}\):

\[u_m(t) = \sqrt{\frac{2\mathcal{E}_s}{T}}\cos\!\left(2\pi f_c t + \frac{2\pi m}{M}\right), \quad 0 \le t \le T\]

2D Geometric Representation

\[\mathbf{s}_m = \left(\sqrt{\mathcal{E}_s}\cos\frac{2\pi m}{M},\; \sqrt{\mathcal{E}_s}\sin\frac{2\pi m}{M}\right)\]

Orthonormal basis functions:

\[\psi_1(t) = \frac{1}{\sqrt{\mathcal{E}_s}}\,g_T(t)\cos(2\pi f_c t), \quad \psi_2(t) = -\frac{1}{\sqrt{\mathcal{E}_s}}\,g_T(t)\sin(2\pi f_c t)\]

Note: BPSK (\(M=2\)) is identical to BASK — the signals are antipodal!

QPSK (M = 4)

Four phases at \(0°, 90°, 180°, 270°\). Energy per bit: \(\mathcal{E}_b = \mathcal{E}_s / \log_2 M\).

Gray Encoding

Key idea: Adjacent phase points differ by only 1 bit. Since noise most commonly pushes a symbol to an adjacent point, this minimizes the number of bit errors per symbol error.

Euclidean Distance

\[d_{mn} = \sqrt{2\mathcal{E}_s\!\left(1 - \cos\frac{2\pi(m-n)}{M}\right)}\] \[\boxed{d_{min} = 2\sqrt{\mathcal{E}_s}\sin\frac{\pi}{M}}\]

For large \(M\): \(\sin(\pi/M) \approx \pi/M\), so \(d_{min} \approx 2\pi\sqrt{\mathcal{E}_s}/M\).

Error Probability

\[P_2 = Q\!\left(\sqrt{\frac{2\mathcal{E}_b}{N_0}}\right), \quad P_4 \approx 2Q\!\left(\sqrt{\frac{2\mathcal{E}_b}{N_0}}\right)\] \[\boxed{P_M \approx 2Q\!\left(\sqrt{\frac{2\pi^2(\log_2 M)\,\mathcal{E}_b}{M^2\,N_0}}\right) \approx 2Q\!\left(\frac{d_{min}}{\sqrt{2N_0}}\right)}\]

Key insight: Doubling \(M\) (adding 1 more bit per symbol) requires +6 dB more signal energy to maintain the same \(d_{min}\).

WORKED EXAMPLE 7

For \(M = 2, 4, 8\) PSK with the same \(\mathcal{E}_s\):

M\(d_{min}\)
2\(2\sqrt{\mathcal{E}_s}\)
4\(\sqrt{2\mathcal{E}_s}\)
8\(2\sqrt{\mathcal{E}_s}\sin(\pi/8) = \sqrt{0.586\,\mathcal{E}_s}\)

To give \(M=8\) the same \(d_{min}\) as \(M=4\): increase \(\mathcal{E}_s\) by factor \(2/0.586 = 3.41 = 5.3\) dB.

4.3 Quadrature Amplitude Modulation (QAM)

🧠 Intuition: PSK constrains points to a circle (constant amplitude). What if we remove that constraint? We can independently modulate both amplitude and phase — or equivalently, put PAM on both the cosine and sine carriers. This gives us a 2D grid that packs signal points more efficiently.

\[u_m(t) = A_{mc}\,g_T(t)\cos(2\pi f_c t) - A_{ms}\,g_T(t)\sin(2\pi f_c t), \quad m = 1, 2, \ldots, M\]

Example: 16-QAM = 4-PAM on \(\cos\) carrier + 4-PAM on \(-\sin\) carrier → 16 points in a 4×4 grid.

Combined amplitude/phase form: \(u_{mn}(t) = A_m\,g_T(t)\cos(2\pi f_c t + \theta_n)\)

Geometric Representation (same basis as PSK)

\[\mathbf{s}_m = \left(\sqrt{\mathcal{E}_s}\,A_{mc},\; \sqrt{\mathcal{E}_s}\,A_{ms}\right)\]

Average Energy

\[\mathcal{E}_{av} = \frac{1}{M}\sum_{m=1}^{M}||\mathbf{s}_m||^2\]

Error Probability (Square QAM)

\[\boxed{P_M \le 4\,Q\!\left(\sqrt{\frac{3\,k\,\mathcal{E}_{b,av}}{(M-1)N_0}}\right), \quad k = \log_2 M}\]

QAM vs. PSK

Comparison factor (ratio of Q-function arguments squared):

\[\mathcal{R}_M = \frac{3/(M-1)}{2\sin^2(\pi/M)}\]

For \(M = 4\): \(\mathcal{R}_4 = 1\) (identical performance). For \(M > 4\): QAM wins — it needs less power for the same error rate.

WORKED EXAMPLE 8

Compare two 8-point constellations with adjacent point separation \(d = 2\):

4.4 M-ary Frequency Shift Keying (M-FSK)

🧠 Intuition: Each symbol uses a completely different frequency. Since the frequencies are chosen to be orthogonal, each one occupies its own dimension in signal space. M-FSK is inherently M-dimensional.

\[u_m(t) = \sqrt{\frac{2\mathcal{E}_s}{T}}\cos\!\left(2\pi(f_c + m\Delta f)t\right), \quad m = 0, 1, \ldots, M-1\]

With \(\mathcal{E}_s = k\mathcal{E}_b\), \(T = kT_b\), and minimum frequency separation \(\Delta f\).

Geometric Representation (M-Dimensional)

\[\mathbf{s}_0 = (\sqrt{\mathcal{E}_s}, 0, \ldots, 0), \quad \mathbf{s}_1 = (0, \sqrt{\mathcal{E}_s}, \ldots, 0), \quad \ldots \quad \mathbf{s}_{M-1} = (0, \ldots, \sqrt{\mathcal{E}_s})\]

Distance between any pair: \(d = \sqrt{2\mathcal{E}_s}\) — all equally spaced!

Correlation coefficient: \(\gamma_{mn} = \mathrm{sinc}(2\pi(m-n)\Delta f T)\)

Detection Methods

  • Coherent: Estimate carrier phases, perform phase-coherent demodulation. Better performance.
  • Non-coherent: Use envelope detectors, ignore phase. Simpler but slightly worse.
⚡ UNIQUE PROPERTY OF M-FSK

Unlike PAM, PSK, and QAM where increasing \(M\) hurts power efficiency (points crowd together), M-FSK gets MORE power efficient as \(M\) increases! Required SNR/bit decreases with \(M\).

The trade-off: bandwidth grows linearly with \(M\) (each frequency needs its own spectral slot).

🧪 Knowledge Check — 10 Questions

⚠ Common Misconceptions & Exam Traps

  1. Confusing antipodal vs. orthogonal geometry. Antipodal = 1D, signals are negatives. Orthogonal = 2D, signals are perpendicular. Their \(P_E\) formulas have different arguments!
  2. Forgetting the 3 dB advantage. Antipodal needs half the power of orthogonal for the same \(P_E\). The reason: \(d_{12}^2 = 4\mathcal{E}_b\) vs. \(2\mathcal{E}_b\).
  3. Mixing up \(\mathcal{E}_b\) and \(\mathcal{E}_s\). \(\mathcal{E}_s = k\mathcal{E}_b = (\log_2 M)\mathcal{E}_b\). Always check which one the formula or problem uses!
  4. Bandpass energy for M-ASK. When \(f_c \gg W\), bandpass energy = half of baseband energy. Don't forget the factor of \(1/2\)!
  5. PSK has equal energies; QAM does not. PSK points lie on a circle → all have energy \(\mathcal{E}_s\). QAM points scatter across a grid → different energies → you need the average.
  6. M-FSK: dimensions grow with M. PSK/QAM always use 2 dimensions. PAM uses 1. But M-FSK uses \(M\) dimensions — one per frequency!
  7. M-FSK gets better with M, not worse. For PSK/QAM, increasing \(M\) hurts power efficiency. For FSK, it helps — at the cost of bandwidth.
  8. Gray coding minimizes BIT errors, not symbol errors. Symbol error rate stays the same; Gray coding just ensures each symbol error causes fewer bit errors.
  9. \(d_{min}\) determines performance. The minimum distance between signal points controls \(P_E\), not individual signal energies.
  10. The orthonormal basis is NOT unique. But geometric properties (distances, angles, energies) are invariant — they don't depend on which basis you choose.

For a given set of signal waveforms: (a) Determine the dimensionality and basis functions. (b) Represent the waveforms as vectors. (c) Find the minimum distance between any pair of vectors.

Hint: Follow the Gram-Schmidt procedure step by step. Check if any \(d_k(t) = 0\) to determine \(N\). Then compute all pairwise distances \(||\mathbf{s}_m - \mathbf{s}_n||\) and find the minimum.

Problem 2: Binary PAM System Design

A binary PAM system uses rectangular pulses with \(R = 10^5\) bits/sec. Given \(N_0 = 10^{-2}\) W/Hz, find the amplitude \(A\) needed for \(P_E = 10^{-6}\).

Hint: For binary PAM, \(P_E = Q(\sqrt{2\mathcal{E}_b/N_0})\). Use \(Q^{-1}(10^{-6}) \approx 4.75\). Remember \(\mathcal{E}_b = A^2 T_b\) and \(T_b = 1/R_b\).

Problem 3: Repeater Chain

A system has 100 digital repeaters in series. Binary antipodal signals are used. If the overall end-to-end \(P_E = 10^{-6}\), find each repeater's \(P_E\) and the required \(\mathcal{E}_b/N_0\).

Hint: For independent errors: \(1 - P_{total} = (1-P_{rep})^{100}\). For small probabilities: \(P_{rep} \approx P_{total}/100 = 10^{-8}\).

Problem 4: 8-QAM Power Comparison

Compare two 8-point QAM constellations with minimum adjacent distance \(d = 2A\). Determine the average transmitted power for each and identify which is more power efficient.

Hint: Compute each point's distance from the origin, square it, and average over all points. The constellation with lower average power is more efficient.

import numpy as np
import matplotlib.pyplot as plt
from scipy.special import erfc

def qfunc(x):
    return 0.5 * erfc(x / np.sqrt(2))

# Simulation parameters
num_bits = 1_000_000
EbN0_dB = np.arange(0, 12, 1)

ber_sim = []
ber_theory = []

for snr_db in EbN0_dB:
    EbN0 = 10**(snr_db / 10)

    # Generate random bits and BPSK symbols
    bits = np.random.randint(0, 2, num_bits)
    symbols = 2 * bits - 1  # 0 -> -1, 1 -> +1

    # Add AWGN noise
    noise_std = 1 / np.sqrt(2 * EbN0)
    noise = noise_std * np.random.randn(num_bits)
    received = symbols + noise

    # Detect: threshold at 0
    detected = (received > 0).astype(int)

    # Count errors
    errors = np.sum(bits != detected)
    ber_sim.append(errors / num_bits)
    ber_theory.append(qfunc(np.sqrt(2 * EbN0)))

plt.figure(figsize=(8, 5))
plt.semilogy(EbN0_dB, ber_theory, 'b-', label='Theory')
plt.semilogy(EbN0_dB, ber_sim, 'ro', label='Simulation')
plt.xlabel('Eb/N0 (dB)')
plt.ylabel('Bit Error Rate')
plt.title('BPSK BER in AWGN Channel')
plt.legend()
plt.grid(True, which='both', alpha=0.3)
plt.ylim([1e-6, 1])
plt.show()
