Geometric Representation of Signal Waveforms
Foundation

📡 1.1 Introduction — Why Digital Modulation?

▼

After A/D conversion, digital signals must be transmitted over AWGN channels — the simplest (yet fundamental) model for both wireline and wireless channels. The core challenge: digital information must be mapped into analog signal waveforms.

Baseband Channel

Passband includes \(f = 0\). No carrier needed. Examples: coaxial cable, twisted pair.

Bandpass Channel

Passband far from \(f = 0\). Carrier modulation required. Examples: radio, satellite.

💡 Key Concept

Binary modulation: maps each bit to one of two waveforms \(s_1(t)\) or \(s_2(t)\).
M-ary modulation: transmits \(k\) bits at a time using \(M = 2^k\) distinct waveforms.

🔬 1.2 Gram-Schmidt Orthogonalization

▼
🎯 Physical Intuition

Think of this like decomposing forces into x, y, z components. We take \(M\) signal waveforms and find \(N \le M\) orthonormal basis functions that span the same space.

Step-by-Step Procedure

Step 1 — First basis function
\[\psi_1(t) = \frac{s_1(t)}{\sqrt{\varepsilon_1}}, \qquad \varepsilon_1 = \int_0^T s_1^2(t)\,dt\]
Step 2 — For the k-th signal

Compute projections onto all existing basis functions:

\[c_{ki} = \int_0^T s_k(t)\,\psi_i(t)\,dt, \quad i = 1, \ldots, k-1\]

Subtract projections to get the residual:

\[d_k(t) = s_k(t) - \sum_{i=1}^{k-1} c_{ki}\,\psi_i(t)\]

If \(d_k(t) \neq 0\), normalize:

\[\psi_k(t) = \frac{d_k(t)}{\sqrt{\varepsilon_k}}, \qquad \varepsilon_k = \int_0^T d_k^2(t)\,dt\]

If \(d_k(t) = 0\): no new basis function — the signal is linearly dependent on previous ones.

Step 3 — Continue

Repeat until all \(M\) signals are exhausted → yields \(N \le M\) orthonormal waveforms.

Key Properties

  • \(N\) orthonormal waveforms \(\{\psi_n(t)\}\) form a basis for the \(N\)-dimensional signal space
  • \(N = M\) if and only if all signals are linearly independent
  • Each signal can be represented as: \(s_m(t) = \sum_{n=1}^{N} s_{mn}\,\psi_n(t)\)
  • Coefficients: \(s_{mn} = \int_0^T s_m(t)\,\psi_n(t)\,dt\)
  • Energy preservation: \(\varepsilon_m = \sum_{n=1}^{N} |s_{mn}|^2 = \|\mathbf{s}_m\|^2\)
  • Inner product: \(\int_0^T s_m(t)\,s_n(t)\,dt = \mathbf{s}_m \cdot \mathbf{s}_n\)
⚠️ Exam Trap

The orthonormal basis set is NOT unique! Different orderings of signals produce different bases. However, geometric properties (distances, energies, angles) are invariant.

Worked Example 1

Given: 4 signal waveforms. Find the orthonormal basis.

Solution
  • \(\varepsilon_1 = 2\), so \(\psi_1(t) = s_1(t)/\sqrt{2}\)
  • \(c_{21} = 0 \Rightarrow d_2(t) = s_2(t)\), \(\varepsilon_2 = 2\), \(\psi_2(t) = s_2(t)/\sqrt{2}\)
  • \(c_{31} = 0\), \(c_{32} = -\sqrt{2}\), \(d_3(t) = s_3(t) + \sqrt{2}\,\psi_2(t)\), \(\varepsilon_3 = 1\), \(\psi_3(t) = d_3(t)\)
  • \(c_{41} = \sqrt{2}\), \(c_{42} = 0\), \(c_{43} = 1\), \(d_4(t) = 0\) → No new basis function!

Result: \(N = 3\) dimensions for 4 signals.

Signal vectors:

\[\mathbf{s}_1 = (\sqrt{2},\,0,\,0), \quad \mathbf{s}_2 = (0,\,\sqrt{2},\,0), \quad \mathbf{s}_3 = (0,\,-\sqrt{2},\,1), \quad \mathbf{s}_4 = (\sqrt{2},\,0,\,1)\]

Alternative basis gives: \(\mathbf{s}_1=(1,1,0)\), \(\mathbf{s}_2=(1,-1,0)\), \(\mathbf{s}_3=(-1,1,1)\), \(\mathbf{s}_4=(1,1,1)\)

Binary Modulation Schemes
Core

🔄 2.1 Binary Antipodal Signaling (Binary PAM)

▼
🎯 Physical Intuition

Like flipping a switch — bit 1 sends \(+p(t)\), bit 0 sends \(-p(t)\). The signals are mirror images.

Binary PAM Waveforms
\[s_m(t) = A_m \cdot g_T(t), \quad 0 \le t \le T_b, \quad m = 1,2\] \[A_1 = A, \quad A_2 = -A\]

where \(g_T(t)\) is a rectangular pulse of unit amplitude.

  • Bit rate: \(R_b = 1/T_b\) bits/sec
  • Energy: \(\varepsilon_m = A^2 T_b\) for both waveforms → \(A = \sqrt{\varepsilon_b / T_b}\)
  • 1D Geometric representation: \(s_1 = +\sqrt{\varepsilon_b}\), \(s_2 = -\sqrt{\varepsilon_b}\)
  • Unit energy basis: \(\psi(t) = g_T(t)/\sqrt{T_b}\)
📌 Important

Any antipodal pair has the same geometric representation regardless of pulse shape — only the energy matters!

📻 2.2 Binary ASK (BASK)

▼

BASK is a special case: antipodal baseband signals modulate a carrier \(\cos(2\pi f_c t)\).

BASK Waveforms
\[s_1(t) = p(t)\cos(2\pi f_c t), \quad s_2(t) = -p(t)\cos(2\pi f_c t), \quad 0 \le t \le T_b\]

For rectangular pulse: \(p(t) = \sqrt{2\varepsilon_b / T_b}\)

Basis Function
\[\psi(t) = \sqrt{\frac{2}{T_b}} \cos(2\pi f_c t)\]

Bandpass signals with spectrum centered near \(\pm f_c\). Same 1D geometric representation as Binary PAM.

📐 2.3 Binary Orthogonal Signaling

▼
🎯 Physical Intuition

Instead of being opposite (antipodal), the two signals are perpendicular (orthogonal) in signal space.

  • \(s_1(t)\) and \(s_2(t)\) have equal energy \(\varepsilon_b\) and are orthogonal: \(\int_0^{T_b} s_1(t)\,s_2(t)\,dt = 0\)
  • Two basis functions: \(\psi_1(t) = s_1(t)/\sqrt{\varepsilon_b}\), \(\psi_2(t) = s_2(t)/\sqrt{\varepsilon_b}\)
2D Signal Vectors
\[\mathbf{s}_1 = (\sqrt{\varepsilon_b},\, 0), \qquad \mathbf{s}_2 = (0,\, \sqrt{\varepsilon_b})\]

⏱️ 2.4 Binary PPM (Pulse Position Modulation)

▼
  • Two pulses differ only in time position
  • Orthogonal with equal energies → baseband signal (spectrum centered near \(f=0\))
Signal Vectors
\[\mathbf{s}_1 = (\sqrt{\varepsilon_b},\, 0), \qquad \mathbf{s}_2 = (0,\, \sqrt{\varepsilon_b})\]

Same geometry as binary orthogonal signaling.

🎵 2.5 Binary FSK (Frequency Shift Keying)

▼
BFSK Waveforms
\[s_1(t) = \sqrt{\frac{2\varepsilon_b}{T_b}}\cos(2\pi f_1 t), \qquad s_2(t) = \sqrt{\frac{2\varepsilon_b}{T_b}}\cos(2\pi f_2 t)\] \[f_1 = \frac{k_1}{2T_b}, \quad f_2 = \frac{k_2}{2T_b} \quad \text{(distinct positive integers } k_1, k_2\text{)}\]
  • Frequency separation: \(\Delta f = m/(2T_b)\)
  • Orthogonal and equal energy → same geometric representation as PPM
  • Bandpass signals (spectrum near \(f_1\) and \(f_2\))

Goal: design a receiver to minimize the probability of error \(P_E\).

📊 3.3 Error Probability

▼
Binary Antipodal (PAM / BASK)
\[P_E = Q\!\left(\sqrt{\frac{2\varepsilon_b}{N_0}}\right)\]
Binary Orthogonal (PPM / FSK)
\[P_E = Q\!\left(\sqrt{\frac{\varepsilon_b}{N_0}}\right)\]
⚠️ Critical: 3 dB Advantage

Antipodal signaling requires 3 dB LESS energy than orthogonal for the same \(P_E\). This is because the Euclidean distance between antipodal signals is \(d_{12} = 2\sqrt{\varepsilon_b}\) vs. \(d_{12} = \sqrt{2\varepsilon_b}\) for orthogonal.

General Formula (Any Binary)
\[P_E = Q\!\left(\sqrt{\frac{d_{12}^2}{2N_0}}\right), \qquad d_{12}^2 = \int_0^{T_b} |s_1(t) - s_2(t)|^2\,dt\]

Worked Example 4

Given: Piecewise signals \(s_1(t)\) and \(s_2(t)\). Find BER.

Solution
\[d_{12}^2 = \int_0^{T_b} |s_1(t) - s_2(t)|^2\,dt = \frac{4}{3}T_b\] \[P_E = Q\!\left(\sqrt{\frac{2T_b}{3N_0}}\right)\]

M-ary Digital Modulations
Advanced

📶 4.1 M-ary PAM

▼
  • \(k\)-bit symbols select \(M = 2^k\) amplitudes
  • Symbol rate: \(R_s = 1/T\), Bit rate: \(R_b = k/T = 1/T_b\)
M-PAM Signal
\[s_m(t) = A_m \cdot p(t) = s_m \cdot \psi(t), \quad m = 1,\ldots,M\] \[A_m = (2m - 1 - M)\,A, \quad m = 1,\ldots,M\]

Amplitudes are symmetric about the origin.

Average Energy
\[\varepsilon_{\text{av}} = \frac{A^2\,\varepsilon_p\,(M^2 - 1)}{3}\]

Example M=4: amplitudes \(\{-3A, -A, A, 3A\}\), \(\varepsilon_{\text{av}} = 5A^2T = 5d^2\).

📡 4.2 M-ary ASK

▼

Baseband PAM modulated onto a carrier: \(u_m(t) = s_m(t)\cos(2\pi f_c t)\). This creates DSB-SC AM with bandwidth \(2W\).

⚠️ Exam Trap

Energy in bandpass = HALF of baseband energy (when \(f_c \gg W\)).

Basis Function
\[\psi_c(t) = \sqrt{\frac{2}{\varepsilon_p}}\,p(t)\cos(2\pi f_c t), \qquad N = 1\]
Error Probability
\[P_M = \frac{2(M-1)}{M}\,Q\!\left(\sqrt{\frac{6\,\varepsilon_{\text{av}}}{(M^2 - 1)\,N_0}}\right)\]

In terms of \(\varepsilon_b\):

\[P_M = \frac{2(M-1)}{M}\,Q\!\left(\sqrt{\frac{6(\log_2 M)\,\varepsilon_{b,\text{av}}}{(M^2 - 1)\,N_0}}\right)\]

🔵 4.3 Phase-Shift Keying (PSK)

▼
🎯 Physical Intuition

All signals have the same amplitude but different phases — they sit on a circle in signal space.

M-PSK Waveform
\[u_m(t) = p(t)\cos\!\left(2\pi f_c t + \frac{2\pi m}{M}\right), \quad m = 0,1,\ldots,M-1\]

For rectangular pulse:

\[u_m(t) = \sqrt{\frac{2\varepsilon_s}{T}}\cos\!\left(2\pi f_c t + \frac{2\pi m}{M}\right)\]
  • Energy per bit: \(\varepsilon_b = \varepsilon_s / \log_2(M)\)
  • 2D basis: \(\psi_1(t) = \frac{1}{\sqrt{\varepsilon_s}}g_T(t)\cos(2\pi f_c t)\), \(\psi_2(t) = -\frac{1}{\sqrt{\varepsilon_s}}g_T(t)\sin(2\pi f_c t)\)
  • BPSK = BASK (special case)
2D Signal Vectors
\[\mathbf{s}_m = \left(\sqrt{\varepsilon_s}\cos\frac{2\pi m}{M},\; \sqrt{\varepsilon_s}\sin\frac{2\pi m}{M}\right)\]

QPSK (M = 4)

\(u_m(t)\) for \(m=0,1,2,3\) gives carriers: \(\cos,\; -\sin,\; -\cos,\; \sin\).

💡 Gray Encoding

Adjacent phases differ by only 1 bit → minimizes bit errors (not symbol errors).

Euclidean Distance

Distance Between Signal Points
\[d_{mn} = \sqrt{2\varepsilon_s\!\left(1 - \cos\frac{2\pi(m-n)}{M}\right)}\] \[d_{\min} = 2\sqrt{\varepsilon_s}\sin\frac{\pi}{M}\]

For large \(M\): \(d_{\min} \approx 2\pi\sqrt{\varepsilon_s}/M\)

Error Probability

M-PSK Symbol Error Probability
\[P_2 = Q\!\left(\sqrt{\frac{2\varepsilon_b}{N_0}}\right)\] \[P_4 \approx 2Q\!\left(\sqrt{\frac{2\varepsilon_b}{N_0}}\right)\] \[P_M \approx 2Q\!\left(\sqrt{\frac{2\pi^2(\log_2 M)\,\varepsilon_b}{M^2 N_0}}\right) \approx 2Q\!\left(\frac{d_{\min}}{\sqrt{2N_0}}\right)\]
⚠️ Key Rule

Doubling \(M\) in PSK requires approximately +6 dB/bit more energy for the same error performance.

Worked Example 7

M\(d_{\min}\)
2\(2\sqrt{\varepsilon_s}\)
4\(\sqrt{2\varepsilon_s}\)
8\(2\sqrt{\varepsilon_s}\sin(\pi/8) = \sqrt{0.586\,\varepsilon_s}\)

To match M=4's \(d_{\min}\): must increase \(\varepsilon_s\) by factor \(2/0.586 = 3.41 = 5.3\) dB.
For large \(M\): doubling phases → +6 dB needed (since \(\sin(\pi/M) \approx \pi/M\)).

🟣 4.4 QAM (Quadrature Amplitude Modulation)

▼
🎯 Physical Intuition

Combine amplitude AND phase modulation. Instead of constraining points to a circle (PSK), spread them over a grid → more efficient use of signal space.

QAM Waveform
\[u_m(t) = A_{mc}\,g_T(t)\cos(2\pi f_c t) - A_{ms}\,g_T(t)\sin(2\pi f_c t)\]
  • 16-QAM = 4-PAM on each quadrature carrier
  • Combined form: \(u_{mn}(t) = A_m \cdot g_T(t) \cdot \cos(2\pi f_c t + \theta_n)\)
  • If \(M_1 = 2^{k_1}\), \(M_2 = 2^{k_2}\): transmits \(k_1 + k_2\) bits/symbol
  • Symbol rate: \(R_s = R_b/(k_1 + k_2)\)
  • Same 2D basis as PSK
Geometric Representation
\[\mathbf{s}_m = \left(\sqrt{\varepsilon_s}\,A_{mc},\;\sqrt{\varepsilon_s}\,A_{ms}\right)\] \[\varepsilon_{\text{av}} = \frac{1}{M}\sum_{i=1}^{M}\|\mathbf{s}_i\|^2\]
QAM Error Probability Bounds
\[1 - \left(1 - 2Q\!\left(\sqrt{\frac{3\,\varepsilon_{\text{av}}}{(M-1)\,N_0}}\right)\right)^{\!2} \;\le\; P_M \;\le\; 4Q\!\left(\sqrt{\frac{3k\,\varepsilon_{b,\text{av}}}{(M-1)\,N_0}}\right)\]

QAM vs PSK

Performance Ratio
\[R_M = \frac{(P_M)_{\text{QAM}}}{(P_M)_{\text{PSK}}} = \frac{3}{(M-1)\cdot 2\sin^2(\pi/M)}\]

QAM is superior for \(M > 4\); same at \(M = 4\).

Worked Example 8

ConstellationAvg PowerdB
8-point rectangular67.8 dB
8-point PAM-PSK4.76.7 dB

PAM-PSK is ~1.1 dB more power efficient than rectangular.

🎵 4.5 M-ary FSK

▼
M-FSK Waveform
\[u_m(t) = \sqrt{\frac{2\varepsilon_s}{T}}\cos\!\big(2\pi(f_c + m\Delta f)\,t\big), \quad m = 0,\ldots,M-1\]
  • \(\varepsilon_s = k\varepsilon_b\), \(T = kT_b\)
  • Orthogonal signals → \(M\)-dimensional representation
  • \(\mathbf{s}_0 = (\sqrt{\varepsilon_s},0,\ldots,0)\), \(\mathbf{s}_1 = (0,\sqrt{\varepsilon_s},\ldots,0)\), …
  • \(d = \sqrt{2\varepsilon_s}\) for all pairs
  • Correlation: \(\gamma_{mn} = \text{sinc}(2\pi(m-n)\Delta f\, T)\)

Coherent Detection

Requires phase estimation. Better performance.

Non-coherent Detection

Envelope detection. Simpler but worse by ~1 dB.

🌟 Unique Property of M-FSK

SNR per bit DECREASES as \(M\) increases — less power needed! This is the opposite of PSK/QAM behavior. M-FSK trades bandwidth for power efficiency.

Common Traps & Misconceptions
Exam Prep
⚠️ Trap #1: Antipodal vs Orthogonal Geometry

Antipodal signals are opposite on a line (1D); orthogonal are perpendicular (2D). Their \(P_E\) formulas differ by a factor of 2 inside the Q function. Don't mix them up!

⚠️ Trap #2: The 3 dB Advantage

Antipodal requires 3 dB LESS energy than orthogonal. Reason: \(d_{\min}^2 = 4\varepsilon_b\) (antipodal) vs \(2\varepsilon_b\) (orthogonal).

⚠️ Trap #3: \(\varepsilon_b\) vs \(\varepsilon_s\)

\(\varepsilon_s = k \cdot \varepsilon_b = (\log_2 M) \cdot \varepsilon_b\). Always check whether the formula uses energy per bit or per symbol!

⚠️ Trap #4: Bandpass Energy Halving

For M-ary ASK: bandpass energy = HALF of baseband energy. This comes from \(\cos^2(2\pi f_c t) \to 1/2\) when \(f_c \gg 1/T\).

⚠️ Trap #5: PSK Equal Energy, QAM Not

In PSK, all signals have the same energy (points on a circle). In QAM, signals have different energies (points on a grid).

⚠️ Trap #6: M-FSK Dimensionality

M-FSK uses \(M\) dimensions (one per frequency). This is unique — all other schemes use at most 2D.

⚠️ Trap #7: M-FSK Performance Direction

M-FSK performance IMPROVES with increasing \(M\) (trades bandwidth for power). PSK/QAM performance DEGRADES with increasing \(M\).

⚠️ Trap #8: Gray Coding Target

Gray coding minimizes bit errors, not symbol errors. The symbol error rate doesn't change with bit mapping.

⚠️ Trap #9: \(d_{\min}\) Is King

\(d_{\min}\) determines error performance, not individual signal energies. Two constellations with the same \(d_{\min}\) have similar error rates.

⚠️ Trap #10: Basis Non-Uniqueness

The Gram-Schmidt orthonormal basis is NOT unique. Different orderings yield different bases. But distances, angles, and energies are invariant.

📝 Problem 1: Gram-Schmidt Application

▼

Problem: Given three signals \(s_1(t) = 2\) for \(0 \le t \le 1\), \(s_2(t) = -2\) for \(0 \le t \le 1\), and \(s_3(t) = 2\) for \(0 \le t \le 0.5\), \(s_3(t) = -2\) for \(0.5 < t \le 1\). Apply the Gram-Schmidt procedure.

💡 Hint

Start with \(\varepsilon_1 = \int_0^1 4\,dt = 4\). Then check if \(s_2(t)\) is a scalar multiple of \(\psi_1(t)\). For \(s_3(t)\), compute the projection \(c_{31}\) carefully — the integral of a product of a sign-changing function with a constant may yield interesting results.

📝 Problem 2: BER Comparison

▼

Problem: A system requires \(P_E \le 10^{-5}\). Compare the required \(\varepsilon_b/N_0\) (in dB) for: (a) Binary antipodal, (b) Binary orthogonal, (c) QPSK, (d) 8-PSK.

💡 Hint

Use \(Q^{-1}(10^{-5}) \approx 4.265\). For antipodal: \(2\varepsilon_b/N_0 = (4.265)^2\). QPSK has the same \(\varepsilon_b/N_0\) requirement as BPSK (why?). For 8-PSK, use the \(d_{\min}\) formula.

📝 Problem 3: QAM Constellation Design

▼

Problem: For 16-QAM with minimum distance \(d_{\min} = 2\), calculate: (a) the average energy per symbol, (b) the average energy per bit, (c) the symbol error probability at \(N_0 = 0.5\).

💡 Hint

16-QAM has signal points at \((\pm 1, \pm 1), (\pm 1, \pm 3), (\pm 3, \pm 1), (\pm 3, \pm 3)\) when \(d_{\min} = 2\). Average energy = mean of all \(\|\mathbf{s}_i\|^2\). Remember \(k = \log_2(16) = 4\) bits/symbol.

📝 Problem 4: M-FSK Bandwidth-Power Trade-off

▼

Problem: Explain why M-FSK requires increasing bandwidth but decreasing power as \(M\) increases. If bandwidth doubles from 4-FSK to 8-FSK, estimate the power saving in dB.

💡 Hint

Each new frequency adds a new dimension. More dimensions → more "room" for noise to be distributed → lower probability of error at the same SNR. Think about orthogonal signal space geometry — the angle between any two signals is always 90°, regardless of \(M\).

📝 Problem 5: Signal Space Distances

▼

Problem: Two binary signals are given by \(s_1(t) = A\sin(2\pi t/T_b)\) and \(s_2(t) = A\cos(2\pi t/T_b)\) for \(0 \le t \le T_b\). (a) Are they orthogonal? (b) Find \(d_{12}\). (c) Express the BER. (d) Compare with antipodal signaling using the same energy per bit.

💡 Hint

Compute \(\int_0^{T_b} s_1(t)s_2(t)\,dt = A^2\int_0^{T_b} \sin(2\pi t/T_b)\cos(2\pi t/T_b)\,dt\). Use the identity \(\sin\theta\cos\theta = \frac{1}{2}\sin(2\theta)\). The energy of each signal is \(\varepsilon_b = A^2 T_b/2\).
